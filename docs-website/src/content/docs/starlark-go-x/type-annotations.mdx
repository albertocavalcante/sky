# Type Annotations

starlark-go-x supports parsing Python-style type annotations.

## Syntax

```starlark
def greet(name: str) -> str:
    return "Hello, " + name

def add(a: int, b: int = 0) -> int:
    return a + b
```

## AST Nodes

- `TypeExpr` - Type expression wrapper
- `TypedParam` - Parameter with type annotation
- `DefStmt.ReturnType` - Return type annotation

## Enabling

```go
opts := &syntax.FileOptions{
    TypeAnnotations: true,
}
f, err := opts.Parse(filename, src, 0)
```
