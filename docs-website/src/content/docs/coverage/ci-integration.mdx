# CI Integration

## GitHub Actions

```yaml
- name: Run tests with coverage
  run: |
    skytest --coverage tests/
    skycov --format=cobertura coverage.json > coverage.xml

- name: Upload coverage
  uses: codecov/codecov-action@v4
  with:
    files: coverage.xml
```

## GitLab CI

```yaml
test:
  script:
    - skytest --coverage tests/
    - skycov --format=cobertura coverage.json > coverage.xml
  artifacts:
    reports:
      coverage_report:
        coverage_format: cobertura
        path: coverage.xml
```
