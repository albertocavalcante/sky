## Implementations

| Implementation | Language | Maintainer |
|----------------|----------|------------|
| [starlark-go](https://github.com/google/starlark-go) | Go | Google |
| [starlark-rust](https://github.com/facebook/starlark-rust) | Rust | Meta |
| [starlark (Java)](https://github.com/bazelbuild/bazel/tree/master/src/main/java/net/starlark/java) | Java | Bazel team |

*[starlark-go-x](https://github.com/albertocavalcante/starlark-go-x) is an experimental fork of starlark-go exploring additional features. The goal is to upstream improvements, not fragment the ecosystem.*

## Developer Tools

### Formatters & Linters

| Tool | Description |
|------|-------------|
| **[Sky](https://github.com/albertocavalcante/sky)** | Full toolkit: lint, format, test, coverage |
| [Buildifier](https://github.com/bazelbuild/buildtools) | Official formatter for BUILD files |
| [Starpls](https://github.com/withered-magic/starpls) | Language server for IDE support |

### Documentation

| Tool | Description |
|------|-------------|
| **skydoc** (Sky) | Documentation generator |
| [Stardoc](https://github.com/bazelbuild/stardoc) | Bazel's documentation generator |

### Testing

| Tool | Description |
|------|-------------|
| **skytest** (Sky) | Test runner with assertions and coverage |
| [unittest.bzl](https://github.com/bazelbuild/bazel-skylib/blob/main/lib/unittest.bzl) | Bazel Skylib testing utilities |
